# 🧭《中外消费品购买力测评》研究与内容框架 v1.0

---

## 一、🎯 核心理念：

> “通过跨国对比消费品价格与购买力，揭示全球品牌的定价逻辑与消费者价值结构。”

这个系列不仅是价格比较，更是一次“消费经济学”的实践。
它让人们理解：**为什么同样的商品在不同国家价格差这么大？**
以及：**这种差异背后，反映了什么样的经济结构与品牌策略？**

---

## 二、🧩 总体框架

| 模块          | 目标                         | 数据/方法                                            | 输出形式                   |
| ------------- | ---------------------------- | ---------------------------------------------------- | -------------------------- |
| 1️⃣ 样本选定   | 明确研究品类与国家范围       | 品类列表 + 国家样本                                  | 《测评计划表》             |

## 三、🧠 分析框架（推荐结构）

### STEP 1 ｜价格差异对比

- 同一款商品在日本、中国、美国售价（官方价或平均市场价）
- 以美元/人民币统一口径换算
- 计算相对差异（如“中/日价格比”）

### STEP 2 ｜购买力平价修正（PPP）

- 引入 IMF / OECD 的 PPP 数据
- 把价格除以该国人均购买力系数
  → 得到「真实购买负担比」

### STEP 3 ｜成因分解

| 影响因子 | 指标说明                   |
| -------- | -------------------------- |
| 汇率     | 日元兑人民币走势           |
| 税收     | 进口关税、消费税、增值税   |
| 物流     | 海外运输及关税环节成本     |
| 品牌策略 | 本土溢价 vs 海外形象       |
| 渠道差异 | 官方旗舰、第三方零售、代购 |
| 消费心理 | “日本产=正品/高质”心理价位 |

### STEP 4 ｜结论呈现

- 📊 “一双鞋的跨国旅程”：展示价格构成与中间成本
- 🗺️ “全球消费力地图”：哪个国家买最划算？
- 🧭 “价格差异背后的逻辑”：品牌定位、政策与文化
